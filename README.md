# 3rdPassJOSMRules
Set of JOSM rules for validation plugin suitable for 3rdpass validation. Please use also the [Missing Maps validation on HOT TM (HOT TM Validator Checker)](https://docs.google.com/document/d/1EhOof9iZ5E9yTe64IPqBy8FHSM9hC7j3MyTE9qSzt1Q/edit?usp=sharing) rules which are focused on regular validation.

These rules are throwing warning for:
- Residential area without node with place attribute
- Node with place attribute that is not inside a residential landuse

To add these rules into your JOSM use https://raw.githubusercontent.com/MissingMaps/3rdPassJOSMRules/refs/heads/main/MM_3rdPassValidationRules.mapcss
